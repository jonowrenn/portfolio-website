Overview

A shareable, secure LLM chatbot demonstrating prompt design, OpenAI SDK v1 integration, stateful chat UX, and modern deployment with API key protection and cost management. Shipped as a live, portfolio-ready GPT chatbot with conversation memory and safe usage controls.

Key Features

  • Topic context and conversation history support for multi-turn CS study sessions.
  • One-click web demo with hard usage limits and token/turn caps for public safety.
  • Optional BYOK (Bring Your Own Key) textbox for advanced users.
  • Polished chat interface with retry/undo functionality via Gradio's ChatInterface.
  • Deployed on Hugging Face Spaces with secrets management and usage guardrails.
