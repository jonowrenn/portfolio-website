# Jonathan Wrenn Portfolio

Static portfolio site with no build step.

## Run locally

From this directory:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Deploy free

This project can be deployed as-is on:

- Vercel
- Netlify
- GitHub Pages

Because it is plain HTML, CSS, and JS, no framework setup is required.

## Files

- `index.html` contains the page structure and content
- `styles.css` contains layout and visual styling
- `script.js` sets the footer year
